## Project Overview
This repository contains the complete work for the **"AI Future Directions"** assignment under the theme **"Pioneering Tomorrow’s AI Innovations"**.

It combines:
- Theoretical analysis of Edge AI and Quantum AI  
- Practical Edge AI prototype (waste classification using TensorFlow Lite)  
- AI-driven smart agriculture IoT simulation  


## Task 1 – Edge AI Prototype (Waste Classifier)
- Dataset: TrashNet (15,000 images, 30 classes)  
- Base Model: MobileNetV2 (transfer learning)  
- Final Validation Accuracy: **55.07%**  
- Model converted & optimized with TensorFlow Lite  
- Ready for Raspberry Pi / Android / Coral Edge TPU  

## Task 2 – AI-Driven Smart Agriculture (IoT Concept)
- Sensors: soil moisture, temperature, humidity, sunlight, pH  
- Model: Linear Regression to predict crop yield  
- Data Flow: Sensors → AI Model → Crop Yield Prediction → Farmer / IoT Dashboard  
- Results: MSE and R² included in report  
- Visualizations: scatter plots and diagrams in report
